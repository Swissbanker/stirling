Logout

The initiator signals end of session by sending a Logout message to the acceptor. The acceptor might need to perform Gap Fill operations before terminating the connection.

Example

The initiator sends a Logout message to the acceptor. The acceptor responds with a Logout message and terminates connection. The session is considered to be closed.

Example

The initiator sends a Logout message to the acceptor. The acceptor terminates connection immediately after which the session is considered to be closed.

Example

The initiator sends a Logout message to the acceptor. The acceptor detects a sequence number gap and replies with a ResendRequest message to recover the messages. The initiator resends the messages and waits for acceptor to issue a Logon message. The acceptor issues a Logout message and terminates connection and the session is considered to be closed.
