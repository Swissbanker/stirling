Connection Status

The Heartbeat message is used to monitor the status of the connection. When either end of the connection has not received any data for (HeartBtInt + "some reasonable transmission time") seconds, a TestRequest message is transmitted. It forces a Heartbeat message to be transmitted from the opposing side.

Example

When TestRequest message with TestReqID set to test-id is received, we reply with a Heartbeat message that has TestReqID set to test-id as well.
