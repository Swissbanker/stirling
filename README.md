xTCH
====

xTCH is a library for encoding and decoding protocols similar to NASDAQ OMX
TotalView-ITCH.


Protocol Profiles
-----------------

xTCH supports the following profiles:

- [NASDAQ OMX Nordic Equity TotalView-ITCH 1.86] [1]
